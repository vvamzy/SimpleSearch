SimpleSearch, is a simple program which can be used to avoid the hassle of opening your browser (and wait for it to load) SimpleSearch uses you "Default Browser" to make a quick search using Google search engine(I know privacy is a issue but >90% of people prefer to use google) . This is a initial build which provides a quick google search.

> Double click on Simplesearch.exe to execute.

*This program uses your "Default Browser" to open webpages. So make sure the browser you use as daily driver is set to Default.

In Windows > click start and search Default apps > Web Browser

*Simplesearch.exe shouldn't be moved from the directory doing so the program will not execute.

*SimpleSearch can be run from anywhere even be  pinned to taskbar for easy access.
